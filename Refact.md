# Refact

## Ключевые изменения

- Убран Fog из рендера.
- Tick окончательно закреплён в server-слое: `SwordsGame.server.tick.TickSystem`.
- DayNightCycle замедлен: yaw обновляется реже.
- Grass восстановлен над cobble в генерации (верх столбов снова травяной).
- HUD и debug/info текст переведены на новый декларативный `text.draw(d -> { ... })` синтаксис.

## DSL/Modding

- Удалён внешний `blocks.dsl` из `resources`.
- Block DSL теперь интегрирован прямо в код (`Registry`), исполняется через Groovy JSR223.
- Добавлена точка расширения для моддинга: `Registry.registerScript(String script)`.
- Регистрация блоков и props очищена/упрощена в едином месте.

## Очистка структуры

- Сохранена единая декларативная точка регистрации блоков.
- Пер-блочные классы (`Stone/Grass/Cobble`) удалены, чтобы избежать дублирования и рассинхронизации.
- Код Registry сжат и приведён к более профессиональной модульной структуре (единый runtime DSL + API расширения).
