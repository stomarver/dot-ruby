  ### Controls
| Function | Keys |
|   :---:  | :---:|
| Fullscreen | F4 or Ctrl + Enter |
| Screenshot | F12 or Ctrl + P |
| Movement | Arrows or Edge-Scroll |
| Rotating | Q/E or Scroll |
| Bomb | LMB |
| Reset | Q |
